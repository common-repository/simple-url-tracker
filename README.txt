Simple URL Tracker is an easy to use campaign link generator. Simply enter your parameters and update the page or post to create a trackable link.

== Description ==

Simple URL Tracker allows you to create customized campaign URLs right from within WordPress pages and posts. It generates a custom link built from the campaign parametes you provide. Once this link is shared you will be able to accurately track inbound traffic from your google analytics account.


**Features of Simple URL Tracker include, but are not limited to:**

* You can build customized campaign urls to share.

* More accurately measure your inbound traffic.

* Analyze where to focus your marketing efforts to receive the highest return on investment.




    

== Installation ==

This section describes how to install the plugin and get it working.
1. Upload `Simple URL Tracker` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Simple URL track will now be available on every post and page to generate your custom trackable campaign URL.
